Introduction

La nomenclature produit présentée dans ce document correspond à un code produit composée d'au moins 24 caractères du "pool" suivant : [a-Z][0-9][-][*][/].
Quelques conventions sont également à noter avant de commencer à détailler l'utilisation de la nomenclature.
Chaque valeur 'Autres' correspond à une valeur non existante en base de données qui sera spécifiée dans la description de l'article. Elle apparaîtra comme une suite de tiret (-).
.

Année

Le second champ à être inclus dans la nomenclature est l'année correspondante à l'article. Elle se code sur trois caractères suivant une règle simple: 2abc → abc. Si l'année n'est pas définie, le code N/A sera utilisée.
.

Conclusion

En mettant bout à bout les différents codes, on obtient un code article final théorique d'au moins 24 caractères (sans spécificités ni confectionneurs).

Le code ainsi obtenu se décompose de la manière suivante : g1a1a2a3f1f2p1p2o1o2d1d2d3d4t1e1e2n1m1m2m3c1c2s1s2...*r1r2... avec les correspondances suivantes :
